# GUI 設計書

## 画面一覧

### 1. ダッシュボード (Dashboard)
- **最新大量保有情報（トップ3）**: 最近報告された大きな変動をカード形式で表示。
- **急増銘柄（1ヶ月以内）**: 前月比で保有株数が大きく増加した銘柄のリスト。
- **急減銘柄（1ヶ月以内）**: 前月比で保有株数が大きく減少した銘柄のリスト。

### 2. バケット管理 (Buckets) - 要ログイン
- **バケット一覧**: 作成済みのバケット（銘柄グループ）を一覧表示。
- **バケット作成/編集**: 新しいバケットの作成、名前の変更。
- **銘柄管理**: バケット内への銘柄追加（コード指定）、削除。
- **並べ替え**: ドラッグ＆ドロップまたはボタンによる表示順の変更。

### 3. 大量保有情報一覧 (Whale Watch List)
- **検索/フィルタ**:
  - 純資産 (Net Assets)
  - 発行済株式数
  - 平均出来高
- **一覧表示**: 報告者、保有割合、変動幅などを表示。

### 4. 銘柄詳細情報 (Stock Detail)
- **基本情報**: 銘柄名、コード、最新価格等。
- **大量保有推移グラフ**: 過去3ヶ月〜の保有比率の変化を可視化。
- **履歴データ**: 過去の報告内容をテーブル形式で表示。

### 5. ユーザー設定 (User Settings) - 要ログイン
- **プロフィール設定**: 自己紹介文の編集。
- **通知設定**: 連絡先メールアドレスの登録・変更。
